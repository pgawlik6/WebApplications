# Google Maps Api project
This is WebApplication using Google Maps API. This application is searching indicated addresses and marking them on map. User can download/upload information about markers from/to XML and TXT file. This application is fully responsive.

## Languages
* JavaScript, CSS3, HTML5

## Other technologies
* Google Maps Api, Media Queries, RWD
